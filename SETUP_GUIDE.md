# A股选股平台 - 安装使用指南

## 第一步：安装 Python

1. 打开 https://www.python.org/downloads/
2. 点击黄色的 **Download** 按钮下载
3. 运行安装程序
4. **⚠️ 重要：第一个页面底部，一定要勾选「Add Python to PATH」**
5. 然后点 **Install Now**，等安装完成

![勾选示意](https://docs.python.org/3/_images/win_installer.png)

---

## 第二步：下载程序

1. 打开 https://github.com/simon-de-jose/a-stock-screener
2. 点绿色的 **Code** 按钮
3. 点 **Download ZIP**
4. 下载完成后，右键解压到桌面（或任意位置）

---

## 第三步：启动

1. 打开解压后的文件夹
2. **双击 start.bat**
3. 会弹出一个黑色窗口（命令行），这是正常的
4. 首次运行会自动完成以下步骤（大约 5-10 分钟，请耐心等待）：
   - 创建运行环境
   - 安装必要组件
   - 拉取A股数据
5. 看到「浏览器打开 http://localhost:8000」就说明启动成功了
6. 打开浏览器（Chrome、Edge 都行），地址栏输入：**localhost:8000**，回车

---

## 使用方法

1. 在搜索框输入一只股票名字，比如「洲际油气」，点 **查找**
2. 系统会自动：
   - 把股票名拆成单字（洲、际、油、气）
   - 列出该股票所属的概念板块（天然气、石油开采...）
   - 显示交易所选项（主板、创业板、科创板、北交所）
3. 勾选你感兴趣的标签
4. 点 **匹配**，就能看到所有符合条件的股票
5. 可以打开「最近涨停优先」开关，涨停股会排在前面

## 常见问题

**Q: 黑色窗口可以关吗？**
不能。关了网页就打不开了。用完之后再关。

**Q: 下次怎么用？**
直接双击 start.bat，不用重新安装，几秒钟就能启动。

**Q: 怎么更新数据？**
点页面搜索框旁边的「刷新」按钮，会在后台重新拉取最新数据。

**Q: 显示「未找到 Python」怎么办？**
说明第一步的 Python 没装好。重新安装，注意一定要勾选「Add Python to PATH」。

**Q: 数据拉取失败怎么办？**
可能是网络问题，稍后再试。页面会显示具体错误信息。
