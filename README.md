# A股选股平台

通过概念板块、字匹配、交易所组合筛选，发现相关A股个股。

## 快速开始

### 前提条件
- 安装 [Python 3.10+](https://www.python.org/downloads/)
- **Windows 用户：安装时务必勾选 "Add Python to PATH"**

### 启动

**Windows：** 双击 `start.bat`

**Mac：** 终端运行 `./start.sh`

首次启动会自动安装依赖并拉取A股数据（约5-10分钟），之后每次启动秒开。

启动后打开浏览器访问：**http://localhost:8000**

### 使用方法

1. 输入股票名称（如「洲际油气」），点击查找
2. 系统自动拆字 + 列出所属概念板块
3. 勾选想要的筛选条件
4. 点击「匹配」查看所有符合条件的股票
5. 可开启「最近涨停优先」排序

### 刷新数据

点击搜索框旁的「刷新」按钮，可更新最新行情和板块数据。

## 注意事项

- 数据来源：东方财富（通过 AKShare，免费无需注册）
- 首次拉数据需要耐心（约500个概念板块逐一拉取）
- 之后建议每天刷新一次即可
